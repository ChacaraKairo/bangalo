# Bangalô Design

Site institucional premium para a Bangalô Design, desenvolvido com Next.js, React, TypeScript, Tailwind CSS e React Three Fiber.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Páginas

- Home
- Projetos
- Serviços
- Processo
- Sobre
- Contato

## Deploy

Projeto preparado para deploy em plataformas compatíveis com Next.js, como Vercel.
